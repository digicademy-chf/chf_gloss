==================
Install and config
==================

..  rst-class:: bignums

1.  Install the extension

    Add the package name to your ``composer.json`` or install the package
    manually.

2.  Set up a data folder

    In a blank folder in your page tree, add a ``GlossaryResource``. Further
    data, such as entries, needs to be located in the same data folder.

..  _display-the-glossary:

====================
Display the glossary
====================

To show the glossary on a specific page of your website, simply add the
:guilabel:`Glossary` plugin and set it up to use the glossary in question.
